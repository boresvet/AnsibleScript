Гайд по использованию страшной и жуткой скриптины для обновления по Ansible

!Конкретно эта скриптина опирается на лежащие в соседних папках "Для обновления без сохранения" и "Для работы с конфигами"!

Соответственно для успешного запуска необходимо изменить: 

1) Прописать по инструкции в папке "Для работы с конфигами" список необходимых конфигурационных файлов, И ПРОВЕРИТЬ, так как все остальные файлы будут стёрты

2) Прописать по инструкции в папке "Для обновления без сохранения" всё необходимое для непосредственно обновления

3) Проверить, что в скрипте в этой папке (Update.sh) при запуске правильно заданы файлы с хостами
